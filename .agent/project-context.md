# 🎯 Project Context: Finance App

### 📖 O čom je tento projekt?
Osobná webová aplikácia na komplexné sledovanie financií, investícií a majetku.

### 👥 Cieľová skupina (Target Audience)
Vlastník aplikácie (osobné použitie), zameranie na prehľadnosť, luxusný dizajn a presnosť dát.

### 🛠 Technický Stack
- **Framework:** Laravel 12.x
- **Admin Panel:** Filament PHP 3.2+
- **Styling:** Tailwind CSS 4.x
- **Database:** Štandardná Laravel DB (pravdepodobne MySQL/PostgreSQL podľa .env)
- **Ostatné:** 
    - `spatie/laravel-activitylog` (logovanie zmien)
    - `scheb/yahoo-finance-api` (investície)
    - `barryvdh/laravel-dompdf` (exporty)

### 🎨 Design & Brand Identity
- **Vibe:** Luxusný, moderný, technický, dôveryhodný.
- **Kľúčové prvky:** Dashboardy s grafmi, Net Worth widgety, tmavý režim (Dark Mode) kompatibilita.

### 🚀 Hlavné ciele (KPIs)
- **Presnosť:** 100% korektné finančné výpočty.
- **Rýchlosť:** Okamžitý prehľad o stave majetku.
- **UX:** Intuitívne pridávanie transakcií a správa investičného portfólia.
